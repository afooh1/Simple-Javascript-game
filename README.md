# Simple-Javascript-game
Welcome to GAME 1, a fully responsive multi-page gaming platform designed with player interaction, user authentication, and admin management in mind. Built entirely with HTML, CSS, and JavaScript, this project offers a seamless gaming experience!


📋 Project Overview
This project simulates a simple online gaming system where users can:

Sign up and log in to play games.

Play engaging browser-based games.

Explore information on About and Contact pages.

Admin users can access a basic admin dashboard for managing aspects of the system.

🛠️ Tech Stack
HTML5

CSS3

Vanilla JavaScript (ES6+)

No external libraries or backend servers required — fully front-end based!
